How it works

Four steps, run automatically for every question you ask.

  1. 01

    Ask a question

    Type anything — a quick fact, a comparison, or an open-ended research question.

  2. 02

    Search the web

    Tavily Agent queries live web sources relevant to your question in real time.

  3. 03

    Analyze results

    Sources are read, compared, and filtered down to the ones worth citing.

  4. 04

    Generate an answer

    A cited, streaming answer is written and shown alongside its source cards.
